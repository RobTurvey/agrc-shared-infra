# Naming and Activation Guidance

## Naming
- Prefer `verb-ing + noun` when practical, for example `analyzing-logs`.
- Keep folder name and metadata `name` identical.

## Activation Description
- Begin with `Use this skill when...`.
- Include at least two intent patterns.
- Keep it specific enough to avoid accidental activation.

## Good Example
`Use this skill when the user asks to analyze production logs or needs a frequency-and-severity summary of repeated errors.`

## Anti-Pattern
`General helper for logs.`
