- [ ] Description starts with `Use this skill when...`
- [ ] At least two activation intents are included
- [ ] `scripts/` present only if repeatable logic is needed
- [ ] `references/` includes deep context only
- [ ] `assets/` includes output templates/examples
- [ ] One self-test prompt has been validated
