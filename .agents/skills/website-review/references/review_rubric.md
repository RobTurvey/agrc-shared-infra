# Website Review Rubric

## 1) Rendering and Runtime Health
- First load status is 2xx
- No critical JS runtime errors
- No failed requests on essential assets

## 2) Information Architecture and Messaging
- Exactly one clear `h1`
- Logical heading structure (`h1` -> `h2` -> `h3`)
- Above-the-fold value proposition is explicit
- Primary CTA is obvious and action-oriented

## 3) Link Integrity
- Internal navigation links resolve
- CTA links resolve
- External links are reachable and intentional

## 4) Accessibility Baseline (Heuristics)
- `lang` attribute is set
- Images have meaningful `alt` text
- Interactive controls have accessible names
- Inputs have labels
- Landmark regions (`main`, `nav`, `header`, `footer`) exist

## 5) Mobile Usability
- No horizontal overflow on common phone widths
- Tap target minimum size near `44x44` px for interactive controls
- Text remains readable without overlap/truncation

## 6) Severity Guidance
- **Critical**: Stops user journey, trust, or compliance
- **High**: Major conversion or usability harm
- **Medium**: Noticeable quality issue with workaround
- **Low**: Polish optimization
