# Minimum Layout for agrc-shared-infra

Required for baseline readiness:

- `.agents/skills/`
- `.github/skills/`
- `.agent/profiles/`
- `templates/`
- `scripts/`
- `README.md`

Recommended first wave content:

- One canonical skill in `.agents/skills/<skill-name>/SKILL.md`
- One Copilot entrypoint in `.github/skills/<skill-name>/SKILL.md`
- One helper script in `scripts/` with `--help`
- One template in `templates/` (placeholder is acceptable for v0.1.0)
